# 🛒 Mini Shop

Мини интернет-магазин с корзиной.

## 🚀 Функционал
- Список товаров
- Добавление в корзину
- Подсчет суммы
- Динамическое обновление интерфейса

## 🛠️ Технологии
- HTML
- CSS
- JavaScript

## 📦 Установка
Открыть index.html

## 💡 О проекте
Реализовано управление состоянием и динамический рендеринг.
